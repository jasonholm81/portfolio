Blackjack

Fancy a game of Blackjack? Click the botton below or the project image to be taken to the game. It's you against the dealer. Click the 'hit' button to have your cards dealt. Once you're confident that you have a high enough score to beat the dealer, click 'stand'. The dealer will then automatically continue until they either win or lose. Click 'deal' once the game is over to restart.

This game was built using vanilla JS, HTML and CSS. The game logic was built using JS and the dealer bot was created using an asynchronous function. The app was deployed using Replit.
